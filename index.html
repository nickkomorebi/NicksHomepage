Watches

A personal deal-hunter that tracks listings for specific watch references across eight global marketplaces in real time.

Technical Highlights

  • Python + Flask backend with HTMX — dynamic UI updates without a JavaScript framework
  • Async scraper aggregating listings from 8+ sources: eBay (OAuth 2.0), Chrono24, Reddit, Yahoo Japan Auctions, Mercari JP, and more
  • SQLAlchemy ORM + PostgreSQL for persistent listing storage and run history
  • Google Sheets API integration for data export
  • On-demand and scheduled scraping runs with a live run history dashboard

Space Cats

A fully polished browser-based arcade shooter built entirely in vanilla JavaScript with zero libraries or frameworks.

Technical Highlights

  • Pure vanilla JavaScript + HTML5 Canvas 2D — zero dependencies, all character art drawn programmatically
  • 8-module OOP architecture with a 9-state game state machine and delta-time capped game loop
  • Galaga-style formation AI with Bézier-curve entry animations; 5 unique boss fights across 6 levels
  • 6-tier weapon upgrade system: evolves from single shots to homing bullets and piercing lasers
  • Unified keyboard + touch input layer for identical gameplay on desktop and mobile
  • Persistent leaderboard via a REST API backend
🍷

Wine Journal

A personal wine journal with AI-assisted label scanning to automatically catalog tasting notes and collection data.

Technical Highlights

  • Next.js App Router + React Server Components with TypeScript, Tailwind CSS, and Radix UI
  • Supabase/PostgreSQL with a fully normalized relational schema — brands, regions, and appellations as distinct entities for accurate faceted search
  • AI vision API integration: auto-classifies wine type from label photos with confidence scoring
  • Multi-photo upload pipeline to Supabase Storage with Next.js Image optimization
  • Faceted search by country, type, and varietal; trips feature for tracking winery visits with named attendees
  • Built as a PWA for mobile-first use during tastings
